# 社区行为准则（Code of Conduct）

## 我们的承诺

为了建设开放、协作、可持续的社区，我们承诺为所有参与者提供一个尊重、包容、专业的交流环境。

## 期望行为

1. 以事实和技术为基础进行讨论。
2. 尊重他人时间与背景差异，保持清晰、具体、可执行的反馈。
3. 对问题聚焦，不进行人身攻击或贬损。
4. 在提出反对意见时，同时给出可验证依据或替代方案。

## 不可接受行为

1. 人身攻击、侮辱、骚扰、歧视性表达。
2. 故意制造噪音、刷屏或破坏讨论秩序。
3. 发布明显恶意内容或安全攻击指引。
4. 泄露他人隐私、敏感信息或访问凭据。

## 执行与处理

维护者有权对违反本准则的行为采取措施，包括但不限于：

1. 口头或书面提醒
2. 删除相关内容
3. 临时或永久限制参与权限

## 报告渠道

如遇到不当行为，请在仓库 Issue 中联系维护者，或通过项目维护渠道私下反馈。我们会尽快处理并尽量保护反馈者信息。
